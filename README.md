# Vue 3 docs

> 个人仿照Vue 2 版本的docs 进行学习性质的描述Vue 3的docs

## 指引
- [Vue.js](https://cn.vuejs.org/)

## How to join ? 

```shell

# clone this repo
git clone  https://github.com/veaba/vue-docs.git 

# install dependent
npm install

# dev
npm run dev 

# build
npm run build

```

## about 
- 个人学习Vue 3.0 项目
